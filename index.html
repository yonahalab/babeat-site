식단과 재고를 하나로
스마트한 이유식 관리

아기의 D-day, 다음주까지 필요한 큐브, 주간 캘린더를 한 화면에서 확인하세요. 식단 완료 한 번이면 큐브가 자동으로 소진됩니다.

  • 아기 D-day와 다음주까지 필요한 큐브 한눈에 확인
  • 주간 캘린더에서 매일의 식단 일정 관리
  • 식단 완료 버튼으로 큐브 자동 소진 처리
  • 식사 시간과 사용된 큐브를 빠르게 파악
Smart Calc

선택한 날짜에 필요한
큐브를 자동 계산

원하는 날짜까지 필요한 큐브 수량과 현재 재고를 비교해, 부족한 재료를 미리 알려드려요.

  • 날짜 선택만으로 필요 큐브 자동 계산
  • 항목별 필요 수량과 현재 재고 비교 표시
  • 부족·충분 상태를 한눈에 파악
  • 소비기한 임박 큐브 경고 알림
Nutrition

우리아이 영양 균형
한눈에 확인

매 식사의 탄수화물, 단백질, 비타민·무기질 비율을 시각적으로 보여드려요. 균형 잡힌 이유식을 쉽게 계획하세요.

  • 탄수화물·단백질·비타민 영양 비율 시각화
  • 포함된 큐브의 종류, 수량, 총량 한눈에 표시
  • 식단 완료 시 큐브 재고 자동 차감
  • 날짜별 식단 기록으로 성장 과정 추적
Inventory

꼭 필요한 정보만 모아
내 냉장고와 완벽 싱크

곡물, 고기, 계란, 채소 등 카테고리별로 큐브를 정리하고 재고 현황을 실시간으로 관리하세요.

  • 카테고리별 큐브 필터와 검색 기능
  • 제작일, 재고 수량, 사용 예정 현황 표시
  • 소비기한 임박 큐브 알림으로 식재료 낭비 방지
  • "만들기 전" 큐브로 미리 식단 계획 가능
AI Cube

AI 자동 분류로
빠른 큐브 등록

큐브 이름만 입력하면 AI가 카테고리를 자동으로 감지해요. 번거로운 분류 없이 빠르게 등록하세요.

  • 이름 입력 시 AI가 카테고리 자동 감지
  • 곡물, 고기, 생선, 계란, 채소 등 8가지 분류
  • 슬라이더로 큐브당 용량 간편 설정
  • 재고 수량 직관적 입력
Together

만들 사람, 먹일 사람
모두 함께 관리

아기 프로필을 등록하고 D-day를 자동으로 추적하세요. 파트너를 초대해 이유식 정보를 함께 관리할 수 있어요.

  • 아기 이름, 생년월일, 프로필 사진 등록
  • D-day 자동 계산 및 표시
  • 초대 코드로 파트너와 공동 관리
  • 여러 아기 프로필 전환 지원
Features
이유식 관리, 이렇게 쉬워져요
매일 반복되는 이유식 준비, 베이빗이 체계적으로 정리해 드려요.
🧊

식단 완료로 자동 소진

식단 완료 버튼 한 번이면 사용한 큐브가 자동 차감돼요. 매번 재고를 수동으로 수정할 필요가 없어요.

📊

필요 큐브 자동 계산

원하는 날짜까지 필요한 큐브 수량을 자동으로 계산하고, 부족한 재료를 미리 알려드려요.

🥗

영양 균형 시각화

탄수화물, 단백질, 비타민 비율을 한눈에 확인하고 균형 잡힌 이유식을 계획하세요.

🗂️

냉장고 완벽 싱크

카테고리별 큐브 재고, 소비기한, 사용 예정 수량까지 내 냉장고 상태를 정확히 파악하세요.

🤖

AI 자동 카테고리 분류

큐브 이름만 입력하면 AI가 카테고리를 자동 감지해요. 번거로운 분류 없이 빠르게 등록하세요.

👨‍👩‍👧

파트너와 함께 관리

초대 코드로 파트너를 연결하면 이유식 정보가 실시간으로 동기화돼요. 만들 사람, 먹일 사람 모두 함께.
